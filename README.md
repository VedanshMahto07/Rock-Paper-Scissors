# Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features
- User vs Computer gameplay
- Random computer choice generation
- Live score tracking
- Interactive UI with hover effects

## Built With
- HTML
- CSS
- JavaScript
